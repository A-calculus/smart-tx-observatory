# CHRONOS Documentation Frontend

This is a standalone Next.js frontend for Vercel.

When importing the repository into Vercel, set:

```text
Root Directory: document
Framework Preset: Next.js
Build Command: npm run build
Output Directory: .next
```

The production CHRONOS backend is not deployed here. This app is a CTO-facing project walkthrough that explains architecture, operational evidence, screenshots, and lessons learned from the running Solana/Jito observability system.
